Website Storyboard

Design planning, layout wireframes, and project resources

Mobile Layout (min 600px)

The mobile layout uses a single-column vertical flow. All <div> groups stack top-to-bottom for easy thumb navigation. Navigation links display vertically for easy thumb access.

Desktop Layout (min 950px)

The desktop layout centres all content within a max-width of 1200px. The header uses Flexbox to place the brand on the left and navigation on the right. The skills section uses CSS Grid (2 columns). Cards have enhanced hover effects and generous padding.
